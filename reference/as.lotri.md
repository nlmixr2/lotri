# As lower triangular matrix

As lower triangular matrix

## Usage

``` r
as.lotri(x, ..., default = "")

# S3 method for class 'matrix'
as.lotri(x, ..., default = "")

# S3 method for class 'data.frame'
as.lotri(x, ..., default = "")

# Default S3 method
as.lotri(x, ..., default = "")
```

## Arguments

- x:

  Matrix or other data frame

- ...:

  Other factors

- default:

  Is the default factor when no conditioning is implemented.

## Value

Lower triangular matrix
