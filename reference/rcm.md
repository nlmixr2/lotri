# Use the RCM algorithm to permute to banded matrix

The RCM stands for the reverse Cuthill McKee (RCM) algorithm which is
used to permute the matrix to a banded matrix.

## Usage

``` r
rcm(x)
```

## Arguments

- x:

  A symmetric matrix

## Value

A permuted matrix that should be banded
